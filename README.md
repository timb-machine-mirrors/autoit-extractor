# AutoIT Extractor

View and Extract Resources in an AutoIT Compiled Executable  

# Features

* Supports all AutoIt versions, including AutoHotKey encoded scripts.
* Doesn't execute the target binary like Exe2Aut

# Prequisites

## Need .NET 4.0 Runtime
